面向协议命令与时间的协议包的数据传输协议:

数据传输层:
    数据发送层
    数据接收层
    根据标注的信息实现
    单个内存驻留的协议包的合并与拆分
    这部分交给下层的适配层完成

协议包解析层:
    第一层级:协议集
        协议集由多个协议体组成
    第二层级:协议体
        规定一个协议功能的组合单元
        协议体由多个编号连续的协议包组成
    第三层级:协议包:
        单次协议传输与处理的最小单元
        协议包的数据长度不固定
        协议包按照包编号传输


协议组件暂时难以找到一个固定的格式描述它
这里采用最简化的模式进行描述
